## Maintaining system stability

Always create a system restore point before you change core hardware drivers. Windows creates these backups automatically, but you can also do it manually. Type "Create a restore point" into your Windows start menu and press Enter. Click Create and name your point. If an update causes an issue, the restore point lets you return your computer to its previous state.
